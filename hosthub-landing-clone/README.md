## ğŸ’» Getting Started

### Prerequisites

- Node.js (v12 or higher)
- NPM or Yarn package manager
- Code editor - VS Code with Sass extension recommended

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mirhatfidan/web-development-starter-kit.git
```

2. Navigate to the project directory:
```bash
cd web-development-starter-kit/hosthub-landing-clone
```

3. Install dependencies:
```bash
npm install
```

4. Compile Sass to CSS:
```bash
npm run sass
```
or watch for changes:
```bash
npm run watch
```

5. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local development server (Live Server in VS Code)
