## ğŸ® How to Play

1. **Start Game** - Player X starts first
2. **Take Turns** - Click an empty square to place your mark
3. **Win** - Get three marks in a row (horizontal, vertical, or diagonal)
4. **Draw** - If all squares are filled with no winner
5. **Reset** - Click reset button to start a new game

## ğŸ¨ Game Features

- **3x3 Grid** - Classic Tic-Tac-Toe board
- **Player Turns** - Alternating X and O marks
- **Win Conditions** - 8 possible winning combinations
  - 3 horizontal rows
  - 3 vertical columns
  - 2 diagonal lines
- **Game Status** - Display current player and game result
- **Score Counter** - Track wins for both players
