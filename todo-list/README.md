## ğŸ”§ How It Works

1. **Add Task**: User enters task text and submits
2. **Display**: Task is added to the list and displayed
3. **Toggle Complete**: Click to mark task as complete
4. **Delete**: Remove task from list
5. **Save**: All changes are saved to local storage
6. **Load**: Tasks are loaded when page is refreshed
